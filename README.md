# Superstore Sales Analysis

Data analysis project using Python, Pandas and Power BI.

## Tools Used
- Python + Pandas (data cleaning & analysis)
- Matplotlib & Seaborn (visualizations)  
- Power BI (interactive dashboard)

## Key Insights
- Total Revenue: $2.26M across 2015-2018
- Technology is top category at $827K (37%)
- West region leads at $710K
- 50% revenue growth from 2015 to 2018
